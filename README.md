# 📦 Inventory Management System

A simple yet effective **Inventory Management System** built using **Python** and **Tkinter**, designed to help businesses keep track of their stock, manage items, and streamline product records through a user-friendly interface.

## 🚀 Features

- 🧾 Add new inventory items
- 📋 View all stored items
- 🔍 Search items by name or ID
- 🖊️ Update existing inventory details
- ❌ Delete items from the inventory
- 💾 Persistent storage using SQLite

## 🛠️ Tech Stack

- **Python 3**
- **Tkinter** - for GUI
- **SQLite** - for local database management

## 🧑‍💻 How to Run

1. **Clone the repository**
   ```bash
   git clone https://github.com/Tejas3103/Inventory-Management.git
   cd Inventory-Management
   ```

2. **Run the application**
   ```bash
   python main.py
   ```

> Make sure you have Python 3 installed on your system.
